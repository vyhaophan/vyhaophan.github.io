Publications

  • 2022.11.03
    A Novel License Plate Image Reconstruction System using Generative Adversarial Network
    IEEE
    This paper proposes a two-stage deep learning based algorithm for license plate reconstruction in parking lot management systems. The first stage uses YOLOv4 transfer learning to detect license plate positions, while the second stage employs Pix2Pix GAN to reconstruct poor quality, unrecognizable license plate images into perceptible ones. The system aims to enhance license plate image quality for both human and computer recognition.
  • 2022.08.17
    Intelligent Scene Text Recognition in Streaming Videos
    Springer International Publishing
    This research explores scene text recognition for extracting text from images and videos, which is increasingly important for visual media on social networks. We evaluated modern deep learning architectures like SAST, EAST, and Rosetta on the Vietnamese Vintext dataset by VinAI, comparing their real-time performance in frames per second.
